# Current Scope

The current active phase is skeleton-first. The repository should stay small, readable, and easy to inspect while the first physical and perception vertical slices are built.

First implementation work should prioritize vertical slices that prove one subsystem at a time instead of building broad frameworks in advance.

Immediate future milestones:

1. Camera capture
2. Board calibration
3. Occupancy grid
4. Servo discovery
5. Safe dry-run servo tools
6. Safe single-joint micro-motion

Future modules are placeholders until explicitly activated. Avoid framework-building before physical subsystem tests have produced measured data and working feedback loops.
